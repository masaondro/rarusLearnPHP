Файл createtable.sql содержит скрипт для создания таблиц.

Файл filldb.sql содержит скрипт для заполнения созданных таблиц данными.

Файл allFantazy.sql содержит скрипт, который выводит название книги и ее авторов для жанра “Фантастика”.

Файл mailPopularAuthor.sql содержит скрипт, который выводит автора, написавшего наибольшее количество книг.
